Proyecto

Apoyo técnico y académico para la creación de una base de datos con información descriptiva y cartográfica de suelos de la cuenca del río Caroní

Objetivo

Crear una base de datos integral de información descriptiva y cartográfica de la variable suelos en la cuenca del río Caroní, como apoyo a la gestión y manejo integrado del paisaje.
